RERC-e Local Grant-Writing Guide is an optional app for rural communities.

What happens after installation

- RERC-e opens with a friendly setup screen. You do not need to run a command.
- On first use, RERC-e downloads the Google Gemma 3 1B writing model after you select Download and start. The download is about 0.81 GB.
- Upgrades keep a verified Google Gemma model already stored on the computer.
- The Gemma model, local reference files, and writing stay on this computer. Community facts and current catalog details use public U.S. government websites.

Important limits

RERC-e is a community-built tool. It is not an EPA grant program. It does not decide who can apply, promise funding, or submit an application. Check the funder's page and have a person review every draft.

This installer is not yet digitally signed. Windows may show a safety notice. Installer lifecycle testing ran on the build computer, not a separate clean Windows virtual machine. Download RERC-e only from the henkelpress/rerc-grant-finder GitHub release page.
