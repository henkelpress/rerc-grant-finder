## Microsoft Visual C++ Runtime

RERCie does not redistribute Microsoft Visual C++ runtime DLLs. If the standard x64 runtime is missing, the launcher downloads the current installer from Microsoft at <https://aka.ms/vs/17/release/vc_redist.x64.exe>, verifies its Microsoft Authenticode signature, and runs the installer. Microsoft supplies and licenses that runtime directly to the user.
