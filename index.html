Optional Windows tool

Meet RERCie, your local grant-writing guide.

RERCie helps you review likely matches and start a grant draft with your own notes.

The setup guide walks you through each step. Local writing does not need a command line, account, or AI key. Optional Census facts need internet access. A free Census API key is only needed if the service rate-limits the computer.

RERCie is a community-built companion. It is not an EPA grant program, does not determine eligibility, and does not submit applications.

First use downloads Google Gemma, about 0.81 GB. Check every fact and funding rule before you apply.

Download RERCie for Windows Windows 10 or 11, 64-bit. No command line. This community-built installer is not yet digitally signed, so Windows may show a safety notice. Installer lifecycle testing ran on the build computer, not a separate clean Windows virtual machine. Technical checks: SHA-256 | release QA
