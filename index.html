Optional Windows tool

Meet RERCie, your local grant-writing guide.

RERCie helps you review likely matches and start a grant draft with your own notes.

The setup guide walks you through each step. You do not need a command line, account, or key.

RERCie is a community-built companion. It is not an EPA grant program, does not determine eligibility, and does not submit applications.

First use downloads Google Gemma, about 0.81 GB. Check every fact and funding rule before you apply.

Download RERCie for Windows Windows 10 or 11, 64-bit. No command line. This community-built installer is not yet digitally signed, so Windows may show a safety notice.
